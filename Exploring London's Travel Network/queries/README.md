# Snowflake SQL Project: Transport for London (TfL) Journey Analysis

This project uses Snowflake SQL to analyze public transport journey data provided by Transport for London (TfL).

## Dataset Info
The dataset includes monthly journey data across various transport types in London. Table: `TFL.JOURNEYS`

## SQL Queries

1. **Most Popular Transport Types**
   - Ranks transport types by total journeys.

2. **Emirates Airline Popularity**
   - Finds top 5 months for Emirates Airline usage.

3. **Least Popular Years for Underground & DLR**
   - Returns 5 years with the lowest journey volumes.
